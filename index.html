搭建个人第一篇blog

@Cpyright:https://superior-leo.gitee.io/2021/01/29/da-jian-shu-yu-zi-ji-de-bo-ke/

软件配置

  1. 安装node.js（14.16.1 LTS）

    https://nodejs.org/en/

  2. 启动 Windows PowerShell

    查看node版本

    查看包管理器版本

  3. 安装 git（2.30.0.2）

    https://git-scm.com/downloads

hexo框架安装

  1. 利用镜像源下载安装cnpm

  2. 查看cnpm版本

  3. 安装hexo

  4. 验证是否成功

  5. 遇到问题：cnpm无法加载（系统禁止运行脚本）

    解决办法：https://blog.csdn.net/l_x_cser/article/details/104956657

    成功解决

搭建个人博客

  1. 建立并进入文件夹blog

    E:\blog

  2. 打开blog所在文件夹，右键git bash即可

  3. 生成博客-启动博客

  4. 进入端口地址，可以看到自己的博客，切忌使用Ctrl+C

  5. 新建一个博客

  6. 进入 blog\source\_posts，找到 my-first-blog.md，用 typora 打开，写入内容

  7. 编辑完成后回到 git bash 输入

    清理博客

    生成博客

    预览博客

部署到gitee上

  1. 注册帐号，新建仓库（仓库名与昵称保持一致）

  2. 安装git插件

  3. 修改 blog\_config.yml 文件（用VS Code打开）

  4. 进行远端部署

  5. 刷新仓库

  6. 进入gitee仓库，点击上方服务选择 Gitee Pages，下方强制https最好勾上。

  7. 配置好后点击gitee pages弹出网址即可预览到你的博客。

  8. 遇到问题：

    • hexo d 命令error

      解决办法：https://blog.csdn.net/weixin_36401046/article/details/52940313

      仍无法使用此插件：https://blog.csdn.net/weixin_40161974/article/details/99441501

      再次重新安装插件
